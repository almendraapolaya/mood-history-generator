# Mood-History-Generator
1. **Mood-History-Generator:**

 Analyzing the Collective Vibe Shift

2. **Description:**

This project investigates how major global crises—like the COVID-19 pandemic—affect our personal "background energy" and music choices. By analyzing Spotify’s audio features (Valence, Energy, Acousticness), the goal is to prove that external events change our daily habits and emotional resonance, even when they don't hit us directly.

3. **Tech Stack:**

Language: Python

Data Libraries: Pandas, NumPy

Statistics: SciPy (Hypothesis Testing & Correlation)

Visualization: Matplotlib, Seaborn

Logic: Object-Oriented Programming (OOP)

4. **How to run the Project:**


5. **Main Features:**

Event-Timeline Mapping: Filters Spotify Top 200 data into "Before," "During," and "After" event phases.

Statistical Mood Proving: Uses SciPy T-Tests to determine if shifts in musical happiness (Valence) are statistically significant.

Historical Mood Generator: A recommendation engine that suggests music based on the "collective energy" of a specific historical date.
